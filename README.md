# Eye

An SRE toolkit to help manage multi-cloud infrastructure at one place. 

## Installation

```bash
git clone https://github.com/UturnOSS/Eye
cd backend/ 
pip3 install -r requirements/dev-requirements.txt
cd Eye/
python3 manage.py runserver
```
